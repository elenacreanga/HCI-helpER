Overall Objectives

The objective ot this test is to obtain quality or quantitative of how effective is the chrome extension to a person with color blindness.

  • See how easy it is for a person to interact with the extension.
  • Detect weak point on using the extension and applying the filters accordingly.
  • Evaluate consistency throughout using the extension.
Scenario

User has a type of color blindness and tries to test the app's effectiveness in correcting it.

Tasks

For completing the test scenario, each user was supervised and instructed to do some tasks.

  • Open and install chrome extenstion: 5 minutes;
  • Open a specific link for testing color blindness: 30 sec;
  • Detect an image that he or she does not understand: 2 min;
  • Apply color filters gradually and check if the image becomes clear: 5 minutes
Post-test Questions

These questions could be presented to the user as a questionnaire or verbally asked.

  • What difficulties did you encounter?
  • Did you perceive the difference on one or multiple filters?
  • Did it help you to have the naming for color blindness to bette choose the right filter?
Notes for Test Coordinator
  • Remind the user that you would like them to 'think aloud' so that you can understand what they are thinking, about the task or the product/application;
  • Read the user the scenario for the test;
  • Read the user the tasks he has to complete.
  • Record the user's actions, comments, questions, and body language (write them in the Test Observations section accompanied by video/audio).
  • Don't offer help; let the user attempt to perform the task. If they ask for help reply with: "What do you think you/that would do?" "What do you think that means?".

Test Results for: Product/Application Component

Success Criteria

A successful design has been achieved when:

  • 50% of users manage to complete all steps in time
  • 80% of users identify and correct their type of color blindness using the extension.

Summary

  • All users knew they had a type of color blindness
  • All users identified the type of color blindness they have with the help of the extension
  • 2 out of three users completely managed to distinguish the numbers they did not perceive before.
  • 1 out of three users partially identified the number written in the image, thus some filters might need some adjustment.

Post-test Questions Answers
  • User 1: He did not encounter major difficulties; He perceived the difference only on one filter, managing to distinguish one number and part of the second one.
  • User 2: Found it extremely easy to apply the filters; perceived the difference only on one filter and managed to fully distinguish the number in the image.
  • User 3: Identified the images which she did not understand and switching through the filters managed to perceive the whole number.
